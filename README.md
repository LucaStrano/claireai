## DEV

Requirements:
 - UV (install with brew: `brew install uv`)

clone the repo:

```bash
git clone https://github.com/LucaStrano/claireai.git
```

Setup the environment with UV:

```bash
uv sync
```

Install pre-commit hooks:

```bash
uv run pre-commit install
```
